# 📈 Linear Regression on Medical Insurance Charges Prediction

## 📌 Project Overview

This project demonstrates how to build a **Linear Regression** model to predict **medical insurance charges** based on demographic and health-related information. It includes data preprocessing, visualization, model training, prediction, and evaluation.

## 🎯 Objective

- Predict medical insurance charges.
- Analyze factors affecting insurance costs.
- Evaluate a Linear Regression model.

## 📂 Dataset

**Dataset:** `insurance.csv`

### Features
- Age
- Sex
- BMI
- Children
- Smoker
- Region

### Target
- Charges

**Dataset Size**
- Rows: **1338**
- Columns: **7**

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## 📊 Exploratory Data Analysis

- Dataset overview
- Dataset information
- Smoking status vs charges visualization

## 🔄 Data Preprocessing

- One-Hot Encoding
- Train-Test Split (80:20)

## 🤖 Model

- Linear Regression

## 📈 Evaluation

- MAE: **4181.19**
- RMSE: **5796.28**
- R² Score: **0.7836**

## 📉 Visualization

- Charges by Smoking Status
- Actual vs Predicted Scatter Plot
